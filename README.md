# mosfet-rul-prediction
Research internship project on power MOSFET degradation analysis and Remaining Useful Life (RUL) prediction using NASA thermal overstress aging data.

This repository contains the work carried out during my research internship on power MOSFET degradation analysis and Remaining Useful Life (RUL) prediction.

The project is based on the NASA Thermal Overstress Aging dataset and focuses on the general study of MOSFET degradation through signal analysis, data preprocessing, health indicator construction, and predictive modeling.

The main objective of this work is to better understand the aging behavior of power MOSFETs and to explore methods for estimating their remaining useful life. This repository presents the overall structure of the project, including the main steps of data preparation, degradation analysis, and modeling.

## Main tasks
- Data extraction and preprocessing
- Signal analysis
- Degradation indicator construction
- Remaining Useful Life estimation
- Predictive modeling with machine learning and deep learning approaches

## Tools and libraries

- NumPy
- Pandas
- Matplotlib
- Scikit-learn
- TensorFlow / Keras
